# Getting Started

## Step 1: Start the Metro Server

```bash
yarn start
```

## Step 2: Install dependencies

```bash
yarn install
yarn pods
```

## Step 3: Start Application

### For Android

```bash
yarn android
```

### For iOS

```bash
yarn ios
```
